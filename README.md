# MCP Testing -> Weather Server + Client

This is a simple MCP server that provides weather information for a given location. Used mostly for testing and learning purposes.

Intended to be used with Claude Desktop.

Built using the Model Context Protocol (MCP) SDK. You can find more information on it at:

- [Model Context Protocol (MCP)](https://modelcontextprotocol.io/docs/getting-started/intro)

## Setup

```bash
npm install
```

## Run

```bash
npm run build
npm run start
```

## Claude Desktop

You'll need to have Claude Desktop installed to use this server.
